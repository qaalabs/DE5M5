## Learner Activity

### Open draw.io or Excalidraw
- Design your pipeline architecture

### Include:
- Data sources
- Bronze/Silver/Gold layers
- Technologies (Python, Pandas, Fabric, GitHub Actions)
- Data flow arrows
- Save as PNG or PDF

Keep it simple! Don't over-engineer
